아래 수치는 국내·외 의류 물류 센터 및 3PL 업체 공개 자료·사례 연구를 기반으로 한 참고값입니다. 센터 규모·시스템·상품 특성에 따라 편차가 크므로 절대 기준이 아닌 방향성 참고로 활용하세요.

① 쿠팡 FC 패션의류 피킹 (2024): 개인 UPH 60 유지 압박 보고, 숙련자 100+ 사례. (오마이뉴스, 나무위키 쿠팡풀필먼트 2024)
② 국내 패션·의류 e-커머스 3PL (2024): 파스토·개미창고 등 소량다품종 중심. 수작업 75~130 Pcs/h. 의류는 편의점·신선 대비 처리 강도 낮음. (파스토 블로그, 현장 후기 2024)
③ 국내 아동의류 전문: 소품종 대량(B2B 포함) 95~135, 다품종 소량(B2C) 55~100 Pcs/h. 검수·태깅 포함 시 하한 하락.
④ CJ대한통운·롯데글로벌로지스 (2024): 공개 수치 없음. 이커머스 물류 일반 기준 90~140 Pcs/h 추정. (채용공고 직무 기술서)
⑤ 글로벌 패스트패션 DC (2025): Zara·H&M 수작업 100~160, KION·Vanderlande 자동화 200~400 Pcs/h. (공급업체 백서)
※ 공개 자료·현장 후기·채용공고를 종합한 참고값. 계약 보장 수치 아님. 직접 수집 자료는 항목 추가로 입력하세요.

Supabase 연동 설정
Supabase를 연동하면 팀원 누구나 같은 URL로 접속해 실시간으로 동일한 데이터를 확인할 수 있습니다. 무료 플랜으로 충분합니다.
① Supabase 프로젝트 생성
1. supabase.com 접속 → 무료 회원가입
2. New project 클릭 → 이름 입력 (예: uhp-dashboard) → 비밀번호 설정
3. 프로젝트 생성 완료까지 약 1~2분 대기
② 테이블 생성 (SQL 실행)
Supabase 대시보드 → 좌측 메뉴 SQL Editor → 아래 SQL 붙여넣기 후 실행
create table if not exists uhp_records (
  id text primary key,
  data jsonb,
  updated_at timestamptz default now()
);
alter table uhp_records enable row level security;
create policy "public access" on uhp_records
  for all using (true) with check (true);
③ API 키 입력
Supabase 대시보드 → Settings → API 에서 확인
